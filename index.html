O que são Clients?

(Voltar)

Um cliente na web é como um navegador de internet, como o Chrome ou o Firefox. Ele é usado para acessar e consumir informações e serviços na internet. Os navegadores são programas que usamos para explorar a internet, eles interpretam as páginas da web e as exibem para nós. Ja as aplicações web são programas ou serviços que usamos diretamente no navegador, como redes sociais e e-mails.

O que são Servers?

(Voltar)

Os servidores são computadores que armazenam e entregam informações para os clientes, como páginas da web e e-mails. Usam hardware e software especializados para funcionar de maneira eficiente e segura. Um exemplo são as empresas de hospedagem de sites que fornecem espaço em servidores para que as pessoas possam publicar seus sites na internet.

Linguagens de Programação

(Voltar)

São linguagens que os programadores usam para dar instruções aos computadores. Elas permitem que os computadores entendam o que queremos que eles façam. Existem tanto as linguagens Servers-Side como Client-Side. Linguagens Servers-Side são usadas nos servidores para processar informações antes de enviá-las para o cliente. Exemplos incluem PHP, Java, Python, entre outras. Linguagens Client-Side são usadas no navegador para tornar as páginas da web interativas. Exemplos incluem JavaScript e HTML.

Estrutura básica de um HTML

(Voltar)

O HTML é a linguagem usada para criar páginas da web. Ele usa etiquetas (tags) para estruturar o conteúdo, como títulos, parágrafos e imagens.

Tags e Atributos

(Voltar)

No HTML, as tags são códigos especiais usados para definir a estrutura e o conteúdo de uma página da web. Elas são colocadas entre colchetes angulares, como < tag >. Por exemplo, < h1 > é uma tag usada para criar um título.

Atributos são informações adicionais que podem ser incluídas dentro das tags para fornecer mais detalhes sobre como um elemento deve ser exibido ou funcionar. Por exemplo, a tag de imagem < img > pode ter um atributo "src" que especifica o caminho da imagem.

Entendendo a diferença entre listas ordenadas e não ordenadas

(Voltar)

Uma lista ordenada em HTML é usada quando a ordem dos itens é importante. Os itens são numerados automaticamente, geralmente com números (1, 2, 3, etc.) ou letras (a, b, c, etc.). As listas ordenadas são criadas usando a tag < ol > e os itens da lista são definidos com a tag < li >.

Já as listas não ordenadas são usadas quando a ordem dos itens não importa. Nesse caso, os itens são marcados com símbolos, como pontos ou traços. As listas não ordenadas são criadas com a tag < ul > e os itens da lista também são definidos com a tag < li >.
